# Contributing 🚀

First off, thanks for taking the time to contribute! It’s people like you who make this library better.

**Repository:** [github.com/kirilinsky/react-calendar-datetime](https://github.com/kirilinsky/react-calendar-datetime)

---

## 🛠 Scripts

Basic commands for development and building:

- `npm run start` — Launches **Ladle** to preview components and themes in isolation.
- `npm run dev` — Starts **tsup** in watch mode for active development.
- `npm run build` — Compiles the library into the `dist` folder for production.
- `npm run build:demo` — Builds a static version of the Ladle documentation/demo.
- `npm run clean` — Removes the `dist` folder to ensure a fresh build.

---

## 💡 Contribution Rules

1. **Fork** the repo and create your branch from `main`.
2. **Keep it lean:** Avoid adding new heavy packages.
3. **TypeScript:** All code must be strictly typed.
4. **Style:** Follow existing coding patterns and naming conventions.

## 📮 Submitting

- If it's a **bug fix**, feel free to open a PR.
- If it's a **new feature**, please open an Issue first to discuss your idea.
